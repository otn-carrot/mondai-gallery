# mondai-gallery

## 用意するもの
- npm
- git
- MongoDB
## クローン
```bash
git clone https://github.com/pb10001/mondai-gallery.git
```
## インストール
```bash
cd mondai-gallery
npm install
```
## npmパッケージの追加
```bash
npm install -g node-dev
```
## MongoDBの起動
ポートは27017
## 開発用サーバ(バックエンド)
```bash
npm run dev
```
## Vue CLI UI(フロントエンド)
```bash
vue ui
```
